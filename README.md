# sales-prediction-report 

1-Project Title:                                                                                               
Sales Intelligence Report                                                                               

2- Short Description:                                                                                                 
An interactive Power BI dashboard designed to analyze sales performance, budget comparison, and profitability. The dashboard provides clear insights into total sales, profit trends, sales vs budget, AI-generated insights, drill-through analysis, and custom tooltips to support data-driven business decisions.                                                                                               

3-Tech Stack:                                                                                
• Power BI Desktop – Dashboard development and visualization                                                                    
• Power Query – Data cleaning and transformation                                                                           
• Data Analysis Expressions (DAX) – Measures and KPI calculations                                                                     
• Microsoft Excel / CSV – Data preparation                                                                                                  

4- Data Source :                                                                                
Source Website: Kaggle.com                                                 

5- Features:                                                                                                                     
📊 Interactive Dashboard Navigation                                                                                        
• Multi-page dashboard with: Home , Sales Report, Sales vs Budget, AI Visuals, Drill Through, Drill Down, Tooltip                                 
• Navigation buttons for easy page switching.                                                                                             

📈 Sales Performance Analysis                                                                                        
• Tracks: Total Sales , Total Profit , Profit Margin                                                                              
• Provides an overview of business performance.                                                        
                                                              
💰 Sales vs Budget Analysis                                                                            
Compares actual sales with budget targets.                                                                             
Shows variance and achievement percentage.                                                          

🤖 AI Insights                                                                                 
• Uses Power BI AI visuals to identify key factors influencing sales.                                     
                                                     
🔍 Drill Through Analysis                                                                          
•Detailed analysis for selected products, categories, or regions.                                                          

📉 Drill Down Analysis                                                                                        
• Explore data from summary level to detailed level.                                                                       

💡 Custom Tooltips                                                                                                    
• Displays additional insights when hovering over charts.                                   

📌 KPI Summary Cards:                                                                                                 
 Total Sales                                                                                
 Total Profit                                                                                                              
 Budget                                                                                                                
                                                                                             
📊 Visual Analytics                                                                                    
Line Charts                                                                                              
Bar Charts                                                                                                       
Column Charts                                                                                         
Tables                                                                                      
Slicers and Filters
